# Overview

This is a practice project suggested by [Boot.dev](https://boot.dev) to practice using TypeScript and test driven development using Vitest. 


The app is built using node and typescript with the [Pokemon API](https://pokeapi.co/) as the source of data. 

Users can run the app using `npm run dev` which compiles using tsc and starts the CLI. Use `help` to get more info on the avaialable functions.
